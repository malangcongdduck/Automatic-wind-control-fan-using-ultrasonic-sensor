<초음파 센서를 활용한 자동 바람 조절 선풍기 매뉴얼>

1. 이 선풍기는 초음파 센서를 활용해서 제작하였습니다.
   사용자와의 거리를 측정하여 자동으로 바람이 조절되는 선풍기 입니다.

2. 선풍기를 사용하고 싶다면, F5를 누르고 웹 서버에 접속합니다.

3. 정상적으로 웹 서버에 접속했을 때, 화면에는 'fan on'과 'fan off', 두 가지의 버튼이 나타납니다.

4. 'fan on' 버튼을 눌렀을 때, 브래드 보드에서는 다른 색의 LED 3개가 불빛이 들어오고 OLED에서는 'hello user!'이 출력되며 선풍기가 작동합니다. 

5-1. 선풍기와의 거리를 약 30cm이내의 범위에 사용자가 위치한다면, 
      선풍기는 사용자가 더위를 많이 느낀다고 판단하여 강풍을 출력하고, OLED에는 'Set the strong wind!'가 출력됩니다.

5-2. 선풍기와의 거리를 약 30cm-50cm 이내의 범위에 사용자가 위치한다면,
      선풍기는 사용자가 더위를 상대적으로 적게 느낀다고 판단하여 약풍을 출력하고, OLED에는 'Set the weak wind!'가 출력됩니다.

5-3. 선풍기와의 거리를 약 50cm 이상의 범위에 사용자가 위치한다면,
      선풍기는 사용자가 더이상의 바람이 필요하지 않다고 판단. 혹은 사용자가 자리를 떠났다고 판단하여 선풍기를 중지하고, OLED에는 'Stop the fan!'이 출력됩니다.
      선풍기를 중지할 때는 출력을 완전히 제거하지 않고, 절전 모드와 같이 작동하고 있지만 모터를 출력하지 않고 사용자가 선풍기의 범위에 들어오게 되면 사용자와의 거리에 따라 다시 선풍기를 작동합니다.

6. 선풍기를 완전히 종료(Power off)하고 싶다면, fan_off 버튼을 누르세요. 그러면 모든 LED가 불빛이 꺼지고 OLED에서는 창이 모두 채워지며 선풍기를 종료합니다.
